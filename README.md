#（TTA）Terminal Agent Assistant
>Terminal Agent Assistant is a tool for managing multiple terminal-based Agents. It currently supports: claudecode, opencode, hermes, and kimicode. In theory, the software can work with more terminal-based Agents, and users can add their own — but since they haven't been specifically adapted, some features may be missing. As of now, the software offers better management of conversation history and the models used by multiple Agents.
>The software studied and drew inspiration from the following open-source projects: Ghostty and CCswitch.
>Most of this project was built with Codex.
