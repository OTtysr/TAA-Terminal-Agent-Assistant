# (TTA) Terminal Agent Assistant

> Terminal Agent Assistant is a tool for managing multiple terminal-based Agents. It currently supports: Claude Code, OpenCode, Hermes, and KimiCode. In theory, the software can work with more terminal-based Agents, and users can add their own — but since they haven't been specifically adapted, some features may be missing. As of now, the software offers better management of conversation history and the models used by multiple Agents.
>
> The software studied and drew inspiration from the following open-source projects: Ghostty and CCswitch.
>
> Most of this project was built with Codex.

## 中文说明

> Terminal Agent Assistant 是一款用于管理多个终端类 Agent 的工具。目前支持：Claude Code、OpenCode、Hermes 和 KimiCode。理论上，这款软件可以配合更多终端类 Agent 使用，用户也可以自行添加，但由于它们尚未经过专门适配，部分功能可能会缺失。目前，这款软件可以更好地管理多个 Agent 的对话历史，以及它们所使用的模型。
>
> 本软件学习并借鉴了以下开源项目：Ghostty 和 CCswitch。
>
> 本项目的大部分内容由 Codex 构建。
